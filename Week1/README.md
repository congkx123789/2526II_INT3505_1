# Week 1 - Giới Thiệu API & Bài Tập Phân Tích

## 📋 Mục tiêu buổi học
- Hiểu được API là gì và tại sao nó quan trọng.
- Biết cách đọc tài liệu API (API Documentation).
- Hình dung được "hình hài" thực tế của một API.
- Tự phân tích ít nhất **3 API công khai** theo khung chuẩn.

## 📁 Cấu trúc thư mục
```
Week1/
├── README.md                    ← File này
├── api_analysis_template.md     ← Khung phân tích chuẩn (điền vào khi làm bài)
├── example_openweather.md       ← Ví dụ mẫu thầy đã phân tích sẵn (OpenWeather)
└── suggested_api_list.md        ← Danh sách API gợi ý để lựa chọn
```

## 📝 Bài tập: Tìm và phân tích 3 API công khai
1. Chọn **3 API** từ danh sách gợi ý (hoặc tự tìm nếu muốn).
2. Dùng `api_analysis_template.md` làm khung, điền đủ thông tin cho từng API.
3. Nộp kết quả dưới dạng file **Markdown** (`.md`) hoặc trên **Notion**.

## 📅 Tiến độ
- **Buổi 1** → Làm bài tập cá nhân / nhóm.
- **15 phút đầu Buổi 2** → Trình bày: 2-3 học viên được gọi random lên chia sẻ API thú vị nhất mình tìm được.
